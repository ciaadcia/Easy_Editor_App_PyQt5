# Aplikasi Easy Editor  
Aplikasi pengolah gambar sederhana berbasis PyQt5 yang memungkinkan pengguna membuka, melihat, dan mengedit gambar dengan beberapa fitur dasar seperti rotasi, mirror, grayscale (hitam-putih), dan peningkatan ketajaman.

Cara penggunaan:

1. Klik tombol Folder untuk memilih folder yang berisi gambar.
2. Pilih file gambar dari daftar yang muncul di sebelah kiri.
3. Gambar akan ditampilkan di area utama.
4. Gunakan tombol:
   - Left untuk memutar gambar ke kiri
   - Right untuk memutar gambar ke kanan
   - Mirror untuk membalik gambar
   - Sharpness untuk meningkatkan ketajaman gambar
   - B/W untuk mengubah gambar menjadi hitam-putih
   - Hasil edit akan otomatis disimpan di folder Modified di dalam folder yang dipilih.

Aplikasi ini cocok untuk editing gambar sederhana dengan tampilan yang mudah digunakan.
